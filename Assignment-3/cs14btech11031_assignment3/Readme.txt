Stop-and-Wait Protocol :
------------------------

Folder : Stop-and-Wait
Contents : sw_server.py & sw_client.py

Running the codes :

In one terminal tab, run :

    python sw_server.py [FILENAME]

Then, in another terminal tab, run :

    python sw_client.py

Make sure that the file whose name is given by FILENAME is in the Stop-and-Wait folder.

Go-back-N Protocol :
--------------------

Folder : Go-back-N
Contents : gbn_server.py & gbn_client.py

Running the codes :

In one terminal tab, run :

    python gbn_server.py [FILENAME] [N]

Then, in another terminal tab, run :

    python gbn_client.py

Make sure that the file whose name is given by FILENAME is in the Go-back-N folder. N is the window size.
